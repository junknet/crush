用 AI 子智能体在**公开互联网**上抓取 URL 或搜索网页，能提取、总结并回答关于**网页内容**的问题；比 `Fetch` 慢且成本高。

**仅限公网。** 本工具只会发起网络请求/网页搜索，**无法**访问本地文件、当前项目、代码库、用户记忆或对话历史。

**禁止用于**（这些用本地工具）：
- 「这个项目/仓库是干嘛的」「找某个名字的项目/文件」→ 用 `ReadDir`/`Read`/`Grep`/`Find`/`Batch`，不要拿项目名去搜网页。
- 查用户历史上下文、文档、记忆 → 读本地文件 / memory，不是 `websearch-agent`。
- 只有当你**确实需要互联网上的外部信息**（公开文档、库用法、报错、新闻、第三方 API 说明）时才用本工具。

**选择规则**：
- 需要理解/提取/回答**网页**内容 → 用 `websearch-agent`（本工具）
- 只需要原始文本或 API 响应 → 用 `Fetch`（更快、更省）
- 需要保存文件 → 用 `Download`
- sub-agent 内读网页 → 用 `web_fetch`
